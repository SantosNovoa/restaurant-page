## How to Run Locally

1. Clone the repo:

```bash
git clone https://github.com/your-username/whiskers-and-brews.git
cd whiskers-and-brews
```

2. Install dependencies:

```bash
npm install
```

3. Start the Webpack dev server:

```bash
npm run start
```

4. Open your browser and visit `http://localhost:8080/`
